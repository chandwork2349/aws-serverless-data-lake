# AWS Serverless Data Lake

## Overview
Built a serverless data pipeline using AWS Glue, S3, and Athena.

## Architecture
Raw Data → S3 → Glue ETL → S3 Curated → Athena

## Technologies
- AWS S3
- AWS Glue
- Athena
- PySpark

## Steps
1. Upload data to S3
2. Run Glue crawler
3. Run Glue ETL job
4. Query using Athena
